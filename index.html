Application Web de Recettes de Cuisine Développée avec Symfony et PostgreSQL

Cette application permet aux utilisateurs de consulter, ajouter, modifier et supprimer des recettes, de laisser des avis et commentaires, ainsi que de gérer des blogs. Elle inclut un CRUD complet pour une gestion dynamique du contenu, avec une interface construite avec Bootstrap et Alpine.js. La gestion de la base de données est réalisée avec PostgreSQL, et l'application est facilement déployable via Docker et Docker Compose.
